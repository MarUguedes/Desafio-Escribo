# Desafio-Escribo

Esse é um algoritimo feito em Dart que , dado um valor, soma todos os elementos que são divisíveis por 3 e 5 e são menores que o valor informado.

<h3>Exemplo</h3>
Caso a função receba o número inteiro 10, ela retorna 23, resultante do somatório
dos números 3, 5, 6 e 9 que são menores que 10.
